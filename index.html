PromptFerret Tools

Open-source AI & D&D utilities

A collection of web tools focused on AI workflows, D&D utilities, and chatbot integrations. Built to remove real friction points. Vibe coded where appropriate. Engineered where necessary.

Available Tools

GSheet Character Exporter

Export Google Sheet D&D 5e character sheets to Markdown or JSON for AI interaction and structured workflows.

SquishText

Compress text into a shareable encoded string and decompress it back. No server, no accounts — runs entirely in your browser.

Encounter Manager

D&D encounter builder and initiative tracker. Create monster templates, build encounters, and run combat with HP tracking, attack rolling, conditions, concentration, and legendary actions — all in your browser.

MarkdownSite

Markdown document browser with hash-based routing and cross-document linking. Drop your .md files alongside the viewer and browse them with working links — supports standard markdown and Obsidian-style wikilinks.
